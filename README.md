# 💰 Money Seconds

> 가만히 있어도 돈은 1초마다 일하고 있다.

이자가 초 단위로 쌓이는 걸 실시간으로 확인해보자 👀

🔗 https://bwjeong.github.io/Money-seconds/

## 기능
- ⏱️ 초·분·시·일 단위 적립 속도 실시간 표시
- 📈 단리 / 복리 선택
- 🧾 세금 15.4%(이자소득세 14% + 지방소득세 1.4%) 토글
- 🎚️ 슬라이더를 이용해 만기까지 임의 시점의 이자 확인
- 💸 만기 시점 최종 이자·금액 계산
